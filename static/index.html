RIPLEY AES 128 Encrypt and Decrypt

A simple REST API for Encrypt and Decrypt

Usage

  • Encrypt 
    • POST https://ripley-aes.herokuapp.com/encrypt
    Request:
    {
    "encrypt":354654654654654
    }
    Response:
    {
    "encrypt": "LIKzZYX4MioOb+C2AFeJaw=="
    }
  • Decrypt
    • POST https://ripley-aes.herokuapp.com/decrypt
    Request:
    {
    "decrypt": "8U216SG4Sgk9WeHF8SeM8Q==" 
    }
    Response:
    {
    "decrypt": "213"
    }
